Accelerated Dynamic Content (AccDc)

AccDC is a free development resource for international businesses, organizations, and academic institutions.

Accdc enables Accessible Innovation within web technologies.

"Accessible Innovation" refers to the creation of technologies that include Accessibility as a core platform feature of development.

The AccDC API is a JavaScript based Dynamic Content Management System that automates the rendering of dynamic content to ensure accessibility for screen reader and keyboard only users.

Keyboard

The tab controls are keyboard accessible:

  • Set focus on the tab control.
  • Press the Arrow keys to move focus between tabs.
  • Press Home or End to move to the first or last tab.
  • Press any letter or number to jump to a tab starting with that character or digit.
  • Press the Spacebar to open (or optionally toggle if configured) the selected tab.
  • Mouse users can click the desired tab as usual.
