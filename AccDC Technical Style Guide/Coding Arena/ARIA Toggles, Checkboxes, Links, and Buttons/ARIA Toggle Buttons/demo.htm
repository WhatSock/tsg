Standard A tag as a toggle:
A Horse!
Standard DIV tag as a toggle:
A Donkey!
Standard SPAN tag as a toggle:
A Goat!

The ARIA toggle button controls are keyboard accessible:

  • Press the Tab key to navigate between the ARIA toggle buttons.
  • Press the Spacebar to toggle the state of the selected toggle button.
  • Mouse users can click a toggle button to toggle the state as usual.
